# 理事長・関係者へURL共有する前の確認リスト

VercelのプレビューURLを共有する前に、以下を確認してください。

## 1. URL確認

- [ ] VercelのURLが開ける
- [ ] トップページが表示される
- [ ] URLが `https://xxxxx.vercel.app` になっている
- [ ] スマホでも同じURLが開ける

## 2. 主要ページ確認

- [ ] トップページ
- [ ] EMIFULLについて
- [ ] サービス一覧
- [ ] 児童発達支援
- [ ] 放課後等デイサービス
- [ ] 保育所等訪問支援
- [ ] 障がい者グループホーム
- [ ] 就労継続支援B型
- [ ] ブランド一覧
- [ ] 施設一覧
- [ ] 相談ページ
- [ ] 採用ページ
- [ ] FAQ

## 3. スマホ確認

- [ ] FVの文字が読みやすい
- [ ] 下部固定CTAが表示されている
- [ ] LINE相談ボタンが押せる
- [ ] 電話相談ボタンが押せる
- [ ] 見学予約ボタンが押せる
- [ ] 施設検索ボタンが押せる
- [ ] ハンバーガーメニューが開く
- [ ] 横スクロールが出ていない
- [ ] 画像のトリミングが不自然ではない

## 4. 相談導線確認

- [ ] 「まずは相談してください」が分かりやすい
- [ ] LINE相談の役割が分かる
- [ ] 電話相談の役割が分かる
- [ ] 見学予約の役割が分かる
- [ ] 施設検索の役割が分かる
- [ ] フォーム相談が見つけやすい

## 5. 仮情報確認

関係者共有前に、仮情報を残したまま共有してよいか確認してください。

- [ ] 電話番号が仮のままではないか
- [ ] LINE URLが仮のままではないか
- [ ] GoogleMap URLが仮検索URLのままでよいか
- [ ] 住所が「確認中」のままでよいか
- [ ] 営業時間が「確認中」のままでよいか
- [ ] 送迎有無が「お問い合わせください」のままでよいか
- [ ] 空き状況が「お問い合わせください」のままでよいか
- [ ] フォーム送信先が仮のままでよいか
- [ ] GA4 IDが仮のままでよいか

## 6. 画像確認

- [ ] FV画像の雰囲気がEMIFULLらしい
- [ ] 各ブランド画像のロゴが統一されている
- [ ] スタッフの服装・ロゴ表現に違和感がない
- [ ] AI感が強すぎない
- [ ] 親子写真ではなく支援風景に見える
- [ ] 関係者確認用として共有して問題ない

## 7. 文言確認

- [ ] 「人生を、咲かせつづける福祉。」が入っている
- [ ] 「わかってもらえた。みとめてもらえた。」の思想が伝わる
- [ ] 福祉用語が多すぎない
- [ ] 保護者が見ても分かりやすい
- [ ] 相談員が見ても支援内容が分かる
- [ ] 求職者が見ても支援観が伝わる

## 8. 技術確認

- [ ] `/sitemap.xml` が開ける
- [ ] `/robots.txt` が開ける
- [ ] 404ページが表示される
- [ ] 下層ページで404にならない
- [ ] 画像が表示される
- [ ] OGP画像が設定されている

## 9. 共有文テンプレート

関係者へ送る文面例：

```txt
EMIFULL障がい福祉事業部 統合サイトのプレビューURLです。
スマホでの見え方を中心にご確認ください。

確認URL：
https://xxxxx.vercel.app

特に見ていただきたい点：
・トップページの印象
・相談導線の分かりやすさ
・各サービスページの内容
・施設ページの見やすさ
・採用ページの支援観

※電話番号、LINE URL、住所、営業時間など一部は公開前の仮情報です。
```

## 10. 共有前の最終判断

- [ ] 仮情報があることを共有相手に伝える
- [ ] 「デザイン確認用」か「公開前最終確認用」かを明記する
- [ ] 修正意見の締切を決める
- [ ] 確認してほしいページを指定する
